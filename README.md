  Análise de Dados - Sample Superstore

 - Sobre o projeto

Este projeto foi desenvolvido como trabalho acadêmico da disciplina **Python, Ciência de Dados e Pandas**, com o objetivo de aplicar técnicas de manipulação, tratamento e análise exploratória de dados utilizando a biblioteca Pandas.

A análise foi realizada utilizando o dataset **Sample Superstore**, contendo informações de pedidos, produtos, clientes, regiões e vendas.

---

 - Objetivos da análise

* Importar e manipular dados utilizando Pandas;
* Explorar a estrutura do conjunto de dados;
* Realizar tratamento de dados faltantes;
* Criar filtros, ordenações e consultas;
* Analisar vendas por categorias e regiões;
* Trabalhar com datas e séries temporais;
* Identificar tendências de vendas ao longo do tempo;
* Criar visualizações gráficas para apoiar a análise.

---

 - Tecnologias utilizadas

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

 - Principais análises realizadas

 - Análise de vendas

* Maiores e menores vendas registradas;
* Categorias com maior quantidade de pedidos;
* Estados e regiões com maior volume de vendas.

 - Análise temporal

* Conversão de datas para o formato `datetime`;
* Extração de ano e mês dos pedidos;
* Evolução das vendas ao longo do tempo;
* Cálculo de variação mensal, crescimento percentual e médias móveis.

 - Cruzamento de dados

* Tabelas cruzadas entre regiões e categorias utilizando `crosstab`;
* Organização e transformação dos dados para facilitar a interpretação.

 - Visualização de dados

Foram criados gráficos para representar:

* Vendas por categoria;
* Vendas por região;
* Evolução das vendas ao longo do tempo.
